# OffshorePro CV Builder — GitHub Pages PWA

## 🚀 Deploy to GitHub Pages (3 steps)

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) → **New repository**
2. Name it `cv-builder` (or anything you like)
3. Set it to **Public**
4. Click **Create repository**

### Step 2 — Upload the file
1. In your new repo, click **Add file → Upload files**
2. Upload `cv_maker_v2.html`
3. **Rename it to `index.html`** (important!)
4. Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. Go to repo **Settings → Pages**
2. Under **Source**, select **Deploy from a branch**
3. Choose **main** branch → **/ (root)**
4. Click **Save**
5. Wait ~1 minute, then your app is live at:
   `https://YOUR-USERNAME.github.io/cv-builder/`

---

## 📱 Installing as a PWA (Add to Home Screen)

### Android (Chrome)
- Open the site in Chrome
- Tap the **3-dot menu → Add to Home screen**
- Or tap the **Install** banner that appears at the bottom

### iPhone/iPad (Safari)
- Open the site in Safari
- Tap the **Share button** (box with arrow)
- Tap **Add to Home Screen**
- Tap **Add**

### Desktop (Chrome/Edge)
- Look for the **install icon** in the address bar
- Or go to Menu → **Install CV Builder**

---

## 💾 Offline Support
The app uses a **Service Worker** to cache itself after the first visit.
All your CVs are saved in **localStorage** and work completely offline.
